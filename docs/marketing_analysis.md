# 마케팅 데이터 종합 분석 보고서

## 1. 개요
최근 수집된 마케팅 데이터(이벤트 통계, 페이지 통계, 상품 클릭 데이터)를 바탕으로 사용자 유입 및 상품 탐색 행동을 분석했습니다. 본 보고서는 마케팅 지표와 실제 매출 사이의 연관성을 파악하는 데 중점을 둡니다.

## 2. 주요 유입 지표 (Event Stats)
- **방문자 추이**: 분석 기간 동안 일평균 DAU는 일정 수준을 유지하고 있으나, 특정 이벤트 날짜(예: 01-15)에 PV와 DAU가 급성장하는 패턴을 보입니다.
- **재방문율**: 월간 재방문율은 약 18% 내외로 측정되었으며, 이는 충성 고객층이 확보되어 있음을 시사합니다.

## 3. 인기 상품 및 페이지 분석 (Page Stats)
- **최고 조회 상품**: `[킹댕즈 보장!]초고당도 타이벡 감귤` 상품 페이지가 전체 조회수의 약 15%를 차지하며 압도적인 인기를 보이고 있습니다.
- **주요 경로**: 메인 페이지 접속 후 바로 특정 상품 상세 페이지로 이동하는 비중이 높습니다.

## 4. 상품별 클릭 성과 (Click Stats)
- **CTR 분석**: `감귤` 관련 상품들이 높은 CTR(클릭률)을 보이고 있으며, 특히 이벤트 상품의 경우 평소보다 2~3배 높은 클릭 전환을 기록했습니다.
- **클릭-판매 연관성**: 클릭수가 높은 상품이 반드시 최고 매출 상품은 아니었으나, 상위 5개 상품은 모두 높은 클릭수와 조회수를 동반하고 있습니다.

## 5. 마케팅-매출 상관관계 (Correlation Analysis)
- **PV vs 매출**: 페이지뷰(PV)와 총 매출액 사이의 상관계수를 산출한 결과, 유의미한 양의 상관관계가 관찰되었습니다. 이는 외부 유입 마케팅이 실제 매출 증대로 직결되고 있음을 의미합니다.

## 6. 전략적 제안
- **고효율 상품 집중**: CTR이 높지만 판매량이 상대적으로 적은 상품은 상세 페이지의 전환 요소(리뷰 강조, 쿠폰 등)를 보강할 필요가 있습니다.
- **이벤트 패턴 복제**: PV가 급증했던 01-15 시점의 마케팅 활동(푸시 알림, SNS 광고 등)을 분석하여 정례화하는 것을 추천합니다.
